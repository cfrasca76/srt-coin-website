# Krügerrand — Wenn Gold, dann richtig.

Animierte One-Page-Website für den Krügerrand (1 oz Goldmünze), inspiriert von den USPs der [SRT Europe GmbH](https://www.srt-europe.de/).

**Features**
- Scroll-gesteuerte Coin-Videosequenz (231 Frames, als Base64 in `index.html` eingebettet — die Seite läuft komplett standalone, einfach `index.html` im Browser öffnen)
- USP-Sektionen mit Parallax-/Blur-Übergängen beim Scrollen
- Shop-Sektion mit Live-Goldpreis pro Unze von [gold-api.com](https://gold-api.com), automatische Aktualisierung alle 3 Minuten
- Der Ordner `frames/` enthält das unkomprimierte Quellmaterial (wird von der Seite nicht benötigt)
